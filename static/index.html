This page shows an implementation of the escape room described in Unreal Engine Escape Room but using the Bevy Rust game engine deployed via web assembly.

Use WASD, space, left shift to move, QE to look left/right, RF to look up/down. F2 toggles mouse-lock, which works on Chrome but not Safari. F1 will pick up an item you are staring at. The inventory panel isn't hooked up yet.

You might have to click on the scene to get keyboard input to work.
